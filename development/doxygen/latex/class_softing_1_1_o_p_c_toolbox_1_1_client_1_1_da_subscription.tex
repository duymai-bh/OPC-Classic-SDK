{\em Gets or sets the requested maximum period between updates sent to the client. Clients can set the keep-\/alive time for a subscription to cause the server to provide client callbacks on the subscription when there are no new events to report. Clients can then be assured of the health of the server and subscription without resorting to pinging the server with calls to Get\+Status.
