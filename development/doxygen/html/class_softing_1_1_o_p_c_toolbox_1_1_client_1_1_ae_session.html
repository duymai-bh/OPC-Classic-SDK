Queries for the filter criteria available in the server's filter space. The filter space is used for the logical structuring of the events. A server can support filtering based on the following values:
- values for the event type(condition related,simple,tracking related)

- values for the event severity
- values for the area (nodes in the eventarea)
- values for the source (leaves in the eventarea)
