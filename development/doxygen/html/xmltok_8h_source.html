179 /* Scan the string starting at ptr until the end of the next complete
180 token, but do not scan past eptr. Return an integer giving the
181 type of token.
182
183 Return XML_TOK_NONE when ptr == eptr; nextTokPtr will not be set.
184
185 Return XML_TOK_PARTIAL when the string does not contain a complete
186 token; nextTokPtr will not be set.
187
188 Return XML_TOK_INVALID when the string does not start a valid
189 token; nextTokPtr will be set to point to the character which made
190 the token invalid.
191
192 Otherwise the string starts with a valid token; nextTokPtr will be
193 set to point to the character following the end of that token.
194
195 Each data character counts as a single token, but adjacent data
196 characters may be returned together. Similarly for characters in
197 the prolog outside literals, comments and processing instructions.
198 */
199
