◆ StringBasedExpiryTimeout

enum Softing.OPCToolbox.Server.StringBasedExpiryTimeout : ulong
Enumerator
DISABLED 

The internal OPC DA Server cache for string-based object data is disabled. Every time the CORE needs any data it will invoke vendor-application callbacks.

INFINITE 

The internal OPC DA Server cache for string-based object data is set to never expire. Data provided once will never be requested again.
