Writes a given message, of a given level, assigned to a given group in the current trace file.

Parameters
levelThe level of tracing. It can be one of the EnumTraceLevel values.
maskThe group to be traced. It can be one of the EnumTraceGroup values.
objectIDThe identifier for the traced object. (it could be the name of the method where the trace message is inserted).
messageThe tracing message. It is a free text and denotes the reason of the trace.
