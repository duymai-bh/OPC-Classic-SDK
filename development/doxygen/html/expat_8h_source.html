460 /* This structure is filled in by the XML_UnknownEncodingHandler to
461 provide information to the parser about encodings that are unknown
462 to the parser.
463
464 The map[b] member gives information about byte sequences whose
465 first byte is b.
466
467 If map[b] is c where c is >= 0, then b by itself encodes the
468 Unicode scalar value c.
469
470 If map[b] is -1, then the byte sequence is malformed.
471
472 If map[b] is -n, where n >= 2, then b is the first byte of an
473 n-byte sequence that encodes a single Unicode scalar value.
474
475 The data member will be passed as the first argument to the convert
476 function.
477
478 The convert function is used to convert multibyte sequences; s will
479 point to a n-byte sequence where map[(unsigned char)*s] == -n. The
480 convert function must return the Unicode scalar value represented
481 by this byte sequence or -1 if the byte sequence is malformed.
482
483 The convert function may be NULL if the encoding is a single-byte
484 encoding, that is if map[b] >= -1 for all bytes b.
485
486 When the parser is finished with the encoding, then if release is
487 not NULL, it will call release passing it the data member; once
488 release has been called, the convert function will not be called
489 again.
490
491 Expat places certain restrictions on the encodings that are supported
492 using this mechanism.
493
494 1. Every ASCII character that can appear in a well-formed XML document,
495 other than the characters
496
497 $@\^`{}~
498
499 must be represented by a single byte, and that byte must be the
500 same byte that represents that character in ASCII.
501
502 2. No character may require more than 4 bytes to encode.
503
504 3. All characters encoded must have Unicode scalar values <=
505 0xFFFF, (i.e., characters that would be encoded by surrogates in
506 UTF-16 are not allowed). Note that this restriction doesn't
507 apply to the built-in support for UTF-8 and UTF-16.
508
509 4. No Unicode character may be encoded by more than one distinct
510 sequence of bytes.
511 */
512 typedef struct
513 {
514 int map[256];
515 void* data;
516 int (XMLCALL* convert)(void* data, const char* s);
517 void (XMLCALL* release)(void* data);
518 } XML_Encoding;
